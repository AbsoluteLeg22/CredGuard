Privacy Policy

This page explains how CredGuard handles information within this educational demonstration project.


Overview

CredGuard is an educational password analysis tool created for demonstration purposes. It is designed to help users understand password strength, weak password patterns and common attack methods.


Password Analysis Data

CredGuard does not aim to permanently store user passwords in plain text. Password analysis is performed to generate strength ratings, crack-time estimates and security suggestions.

For dashboard analytics, limited anonymised metrics such as entropy score, rating and timestamp may be stored.


Admin Access

Access to administrative analytics is protected through an access key, login credentials and temporary lockout controls after repeated failed attempts.


Cookies and Sessions

CredGuard uses session-based functionality to manage administrator access and temporary login state. These sessions help protect restricted areas of the application.


Educational Use

CredGuard is intended as a university cybersecurity project and educational prototype. It should not be treated as a production-grade password management or authentication platform.


Future Enhancements

In a full production system, additional safeguards would typically include a formal cookie notice, stronger audit logging, clearer retention controls and more detailed compliance documentation.
